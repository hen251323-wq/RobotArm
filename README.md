# Robot Arm

Cool robot arm.

## Controls

### Upper Arm

- [W]       Move forward.
- [A]       Rotate left.
- [S]       Move back.
- [D]       Rotate right.

### Lower Arm

- [Up]      Move forward.
- [Left]    Rotate left.
- [Down]    Move back.
- [Right]   Rotate right.

### Camera

- [Space]   Toggle right camera lock.
